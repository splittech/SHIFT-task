# Тестовое задание ШИФТ

**Выполнил:** Степанов Вячеслав Александрович

## Общее Описание

Программа представляет собой CLI-утилиту, которая принимает файлы со записями представляющими различные типы данных (целые числа, дробные числа, строки). Программа сортирует эти записи по выходным файлам в зависимости от типа.
## Флаги

Поддерживается использование как сокращенных, так и полных наименований флагов в любом порядке.

Все опции можно увидеть,  используя флаг --help или -h

| Флаг | Назначение                            | Пример        | Результат примера                                                                                                                                                                                         | По умолчанию                                                                                |
| ---- | ------------------------------------- | ------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| -o   | Путь для выходных файлов              | -o /some/path | Выходные файлы будут созданы в /some/path                                                                                                                                                                 | Выходные файлы будут созданы в текущей папке                                                |
| -p   | Префикс для выходных файлов           | -p result_    | Выходные файлы будут называться: "result_integers.txt" "result_strings.txt"<br>"result_floats.txt"                                                                                                        | Выходные файлы не будут иметь префикса:<br>"integers.txt" "strings.txt"<br>"floats.txt"<br> |
| -a   | Режим добавления в существующие файлы | -a            | Выходные файлы не перезаписываются, результаты добавляются в существующие файлы                                                                                                                           | Выходные файлы перезаписываются                                                             |
| -s   | Краткая статистика                    | -s            | Вывод в консоль количества записей для каждого файла                                                                                                                                                      | Нет вывода в консоль                                                                        |
| -f   | Полная статистика                     | -f            | Вывод в консоль в дополнение к краткой статистике для чисел выводится минимальное и максимальное число, а также сумма и среднее; для строк в дополнение выводится максимальная и минимальная длина строки | Нет вывода в консоль                                                                        |
## Особенности реализации

- В качестве числовых типов были выбраны BigInteger и BigDecimal, что замедляет работу, но практически снимает ограничение на размер для входных данных.
- Для удобства работы с CLI используется дополнительная библиотека [Apache Commons CLI](https://commons.apache.org/proper/commons-cli/)
- При любой ошибке ввода команды, программа сообщает об этом и завершается, не начиная обработку файлов (для удобства пользователя).
- Есть unit-тесты, которые проверяют базовую логику работы.
## Инструкция по запуску

**JDK**: Oracle OpenJDK 25.0.2
**Система сборки:** Maven 3.9.11

Все правила для сборки, включая зависимости представлены в файле **pom.xml**.

Имеется релиз с jar-файлом с собранными зависимостями.

### Пример команды запуска
```bash
java -jar uXl.jar -s -a -p sample- in1.txt in2.txt
```
